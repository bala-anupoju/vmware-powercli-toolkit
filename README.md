# 🖥️ VMware PowerCLI Toolkit

Enterprise VMware vSphere automation scripts —
VM provisioning, snapshots, patching, and reporting at scale.

## 📁 Scripts

| Script | What It Does |
|---|---|
| `bulk-vm-provisioning.ps1` | Deploy hundreds of VMs from template |
| `snapshot-management.ps1` | Create, rotate, delete snapshots at scale |
| `host-health-check.ps1` | ESXi host performance + health reporting |
| `vm-migration.ps1` | vMotion and Storage vMotion automation |
| `capacity-planning.ps1` | Resource usage + growth trend analysis |
| `patch-remediation.ps1` | VUM-based host patching automation |
| `p2v-migration.ps1` | Physical to virtual migration helper |
| `dr-failover-test.ps1` | SRM failover testing automation |
